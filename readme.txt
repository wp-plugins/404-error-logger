== Description ==

This plugin generates a simple log of 404 (Page Not Found) errors on your WordPress site. This can be useful for tracking down broken links or missing content.

It requires PHP version 5 or greater.

If you come across any bugs or have suggestions, please contact me at [rayofsolaris.net](http://rayofsolaris.net).

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin settings can be accessed via the 'Tools' menu in the administration area, under '404 Error Logger'
